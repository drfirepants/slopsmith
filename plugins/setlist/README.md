# Slopsmith Plugin: Setlist Builder

A plugin for [Slopsmith](https://github.com/byrongamatos/slopsmith) that lets you create ordered playlists of songs for gig prep, practice routines, or themed sessions.

## Features

- **Create setlists** with custom names
- **Search and add songs** from your library, choosing which arrangement (Lead/Rhythm/Bass) per song
- **Reorder songs** with up/down buttons
- **Play All** — plays songs sequentially with a floating progress widget showing current position, prev/next/stop controls
- **Auto-advance** — automatically moves to the next song when the current one ends
- Rename and delete setlists

## Installation

```bash
cd /path/to/slopsmith/plugins
git clone https://github.com/byrongamatos/slopsmith-plugin-setlist.git setlist
docker compose restart
```

The "Setlists" link will appear in the navigation bar.
