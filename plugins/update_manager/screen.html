Install and update slopsmith plugins, and keep the slopsmith core itself in sync — all without leaving the browser.

Changes applied

Restart the server so the new code loads. The easiest option is in-place restart below; alternatively run docker compose restart yourself.

docker compose restart

Core update requires container rebuild

Upstream commits touch files that aren't bind-mounted into the container (e.g. Dockerfile, requirements.txt). Pull and rebuild manually on the host:

cd slopsmith && git pull && docker compose build web && docker compose up -d
